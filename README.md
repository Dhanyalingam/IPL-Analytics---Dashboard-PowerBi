IPL Analytics Dashboard (2008-2026)

PROJECT OVERVIEW

This Project is an Interactive Power BI dashboard developed to analyze Indian Premier League(IPL) data from 2008 to 2026.
The dashboard provides insights into team performance,player statistics,batting analysis,and bowling analysis using PowerBI visualization and DAX measures.

 
 ** FEATURES **

** OVERVIEW DASHBOARD **

> Season-Wise IPL Summary

> Champion and Runner up details

> Orange Cap and Purple Cap holders

> Highest fours and sixes statistics

> Points table analysis

** PLAYER ANALYSIS DASHBOARD **

> Total Runs

> Matches Played

> Strike Rate

> Fours and Sixes

> Highest Score

> Runs by Season

> Runs by Oppostion

> Runs by Venue

** BOWLER ANALYSIS DASHBOARD **

> Total Wickets

> Dot Balls

> Economy Rate

> Bowling Average

> Bowling Strike Rate

> Wickets by Season

> Wickets by Oppostion

> Wickets by Venue

> Dismissal Type Analysis

** TOOLS USED **

> Power BI Desktop

> DAX

> Microsoft Excel

> CSV Dataset
